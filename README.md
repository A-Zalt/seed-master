# Seed Master

This mod allows you to easily manipulate your RNG. It's mainly designed for botters/showcasers.

The mod is currently entirely controlled through its settings, which are available from any level's pause menu by default.

Current mod features:
- Set your seed to **any value**
- On checkpoints, the seed is forced to the correct value by default
- Display your seed during gameplay
- Control how the starting seed is generated
